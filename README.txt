ТЕЛЕФОННЫЙ СПРАВОЧНИК (Console Phone Book)

Консольное приложение «Телефонный справочник», реализованное с использованием паттерна MVC (Model–View–Controller). 
Проект предназначен для хранения, просмотра, поиска, создания, изменения и удаления контактов, которые сохраняются в JSON-файле.

============================================================

ВОЗМОЖНОСТИ

Просмотр всех контактов

Поиск контакта по различным полям (имя, телефон)

Добавление нового контакта

Изменение существующего контакта

Удаление контакта по номеру телефона

Хранение данных в JSON-файле

Консольный интерфейс с меню навигации

============================================================

АРХИТЕКТУРА

Проект построен по архитектурному паттерну MVC:

Model — отвечает за работу с данными (чтение/запись JSON, логика контактов) 
View — отвечает за ввод и вывод данных в консоль 
Controller — управляет логикой приложения и связывает Model и View


============================================================

ФОРМАТ ХРАНЕНИЯ ДАННЫХ

Контакты хранятся в файле phone_book.json в формате:

{ 
    "0": 
        { 
            "name": "Ivan Ivanov", 
            "phone": "+71234567890", 
            "comment": "Работа" 
        } 
}

Каждый контакт содержит следующие поля:

name — имя контакта
phone — номер телефона
comment — комментарий

============================================================

МЕНЮ И СООБЩЕНИЯ

Файл menu_message.json содержит:
пункты меню;
названия действий;
навигационные сообщения интерфейса.

============================================================

ЗАПУСК ПРИЛОЖЕНИЯ

Убедитесь, что установлен Python версии 3.10 или выше

Установите зависимости:

pip install -r requirements.txt

Запустите приложение:

py main.py

============================================================

ОСОБЕННОСТИ РЕАЛИЗАЦИИ

Используется dataclass для хранения состояния View

Для оформления вывода применяются декораторы

Проверяется корректность структуры JSON-файла

Обработка ошибок без аварийного завершения программы
